# 七天卡活动系统使用说明

## 系统概述

七天卡活动系统是一个活动配置管理解决方案，支持活动标题、打码倍数、价格配置和奖励配置。系统提供直观的后台管理界面，方便管理员配置七天卡活动的各项参数。

## 功能特性

- **活动配置管理**：支持活动标题、打码倍数、价格配置和奖励配置
- **奖励配置**：支持7天的三种奖励类型配置
- **用户友好**：直观的后台管理界面
- **数据验证**：完整的数据验证和错误提示

## 系统架构

### 数据库表结构

1. **七天卡活动配置表** (`slot_seven_day_card_config`)
   - 活动标题、打码倍数、价格配置
   - 七天奖励、救援金、每日奖励配置

### 核心组件

- `SevenDayCardConfig`：活动配置模型
- `SevenDayCard`：后台管理控制器

## 安装部署

### 1. 数据库初始化

```sql
-- 执行数据库表结构
source database/seven_day_card_tables.sql
```

### 2. 配置路由

在后台管理系统中添加七天卡活动菜单：

```php
// 在菜单配置中添加
[
    'name' => 'seven_day_card',
    'title' => '七天卡活动',
    'icon' => 'fa fa-calendar',
    'children' => [
        [
            'name' => 'seven_day_card_config',
            'title' => '活动配置',
            'url' => '/admin/activity.seven_day_card/config'
        ]
    ]
]
```

### 3. 权限配置

确保管理员有访问七天卡活动相关页面的权限。

## 使用指南

### 活动配置

1. **基本信息配置**
   - 设置活动标题
   - 配置打码倍数
   - 设置活动状态

2. **价格配置**
   - 设置划线价格（原价）
   - 设置现价（实际售价）

3. **奖励配置**
   - 配置7天的七天奖励金额
   - 配置7天的救援金金额
   - 配置7天的每日奖励金额

## API接口

### 后台管理接口

- `GET /admin/activity.seven_day_card/edit` - 获取活动配置
- `POST /admin/activity.seven_day_card/edit` - 保存活动配置

### 响应格式

```json
{
  "code": 1,
  "msg": "操作成功",
  "data": {
    // 具体数据内容
  }
}
```

## 配置示例

### 默认奖励配置

```json
{
  "seven_day_rewards": [22, 5, 7, 4, 4, 4, 8],
  "rescue_rewards": [3, 3, 3, 3, 3, 3, 3],
  "daily_rewards": [1, 1, 3, 1, 1, 1, 5]
}
```

### 价格配置示例

```json
{
  "original_price": 0.00,
  "current_price": 19.99
}
```

## 注意事项

1. **数据安全**：奖励发放会直接影响用户余额，请谨慎操作
2. **时间控制**：确保订单时间范围正确设置
3. **状态管理**：及时更新订单和奖励状态
4. **权限控制**：只有授权管理员才能操作奖励发放

## 故障排除

### 常见问题

1. **奖励发放失败**
   - 检查用户是否存在
   - 检查奖励记录状态
   - 查看数据库连接

2. **统计数据不准确**
   - 刷新统计数据
   - 检查数据完整性

3. **页面加载失败**
   - 检查路由配置
   - 确认权限设置
