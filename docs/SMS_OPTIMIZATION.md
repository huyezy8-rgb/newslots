# 短信验证码接口优化

## 优化概述

对发送短信验证码接口进行了全面优化，提升了代码质量、安全性和用户体验。

## 主要优化内容

### 1. 代码结构优化

**优化前**：
- 大量重复代码
- 逻辑混乱，难以维护
- 缺少参数验证

**优化后**：
- 模块化设计，职责单一
- 消除代码重复
- 清晰的业务逻辑分离

### 2. 新增安全功能

#### 频率限制
- ✅ **1分钟限制**: 同一手机号1分钟内只能发送一次
- ✅ **每日限制**: 每个手机号每天最多发送10次
- ✅ **防刷机制**: 有效防止恶意刷取验证码

#### 参数验证
- ✅ **手机号格式验证**: 支持国际手机号格式
- ✅ **事件类型验证**: 只允许合法的事件类型
- ✅ **参数完整性检查**: 确保必要参数不为空

#### 业务逻辑验证
- ✅ **绑定验证**: 检查账户是否已绑定、手机号是否被占用
- ✅ **登录验证**: 检查手机号是否已注册
- ✅ **忘记密码验证**: 确保手机号存在

### 3. 开发体验优化

#### 测试模式
- ✅ **固定验证码**: 开发环境可使用固定验证码
- ✅ **跳过发送**: 测试时可跳过实际短信发送
- ✅ **日志记录**: 详细的操作日志

#### 配置化
- ✅ **配置文件**: 支持通过配置文件调整参数
- ✅ **环境变量**: 支持环境变量配置
- ✅ **灵活配置**: 过期时间、频率限制可配置

### 4. 错误处理优化

#### 详细错误信息
- ✅ **明确的错误提示**: 用户能清楚了解错误原因
- ✅ **多语言支持**: 中文和英文错误提示
- ✅ **错误分类**: 区分不同类型的错误

#### 日志记录
- ✅ **发送失败日志**: 记录短信发送失败的详细信息
- ✅ **操作日志**: 记录关键操作
- ✅ **错误追踪**: 便于问题排查

## 接口变化

### 请求参数 (无变化)
```json
{
    "mobile": "+1234567890",
    "event": "bind|login|forgot_password"
}
```

### 响应格式 (优化)

**成功响应**:
```json
{
    "code": 1,
    "msg": "短信发送成功",
    "data": {
        "expire_time": 300,
        "message": "验证码已发送到您的手机"
    }
}
```

**错误响应**:
```json
{
    "code": 0,
    "msg": "短信发送过于频繁，请稍后再试",
    "data": null
}
```

## 配置说明

### 短信配置文件 (`config/sms.php`)

```php
return [
    'default' => [
        'expire_time' => 300, // 验证码过期时间
        'rate_limit' => [
            'interval' => 60,    // 发送间隔
            'daily_limit' => 10, // 每日限制
        ],
    ],
    'test_mode' => env('SMS_TEST_MODE', false),
    'test_code' => env('SMS_TEST_CODE', '123456'),
    'skip_send' => env('SMS_SKIP_SEND', false),
];
```

### 环境变量配置

```bash
# .env 文件
SMS_TEST_MODE=true
SMS_TEST_CODE=123456
SMS_SKIP_SEND=true
```

## 新增错误类型

| 错误代码 | 错误信息 | 说明 |
|---------|---------|------|
| 0 | 手机号不能为空 | 未提供手机号 |
| 0 | 事件类型不能为空 | 未提供事件类型 |
| 0 | 手机号格式不正确 | 手机号格式验证失败 |
| 0 | 无效的事件类型 | 不支持的事件类型 |
| 0 | 短信发送过于频繁，请稍后再试 | 触发频率限制 |
| 0 | 今日短信发送次数已达上限 | 触发每日限制 |
| 0 | 保存短信记录失败 | 数据库操作失败 |


### 前端调用示例

```javascript
const sendSms = async (mobile, event) => {
    try {
        const response = await fetch('/api/account/send_sms', {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'Authorization': 'Bearer ' + token
            },
            body: JSON.stringify({
                mobile: mobile,
                event: event
            })
        });
        
        const result = await response.json();
        
        if (result.code === 1) {
            console.log('短信发送成功');
            console.log(`过期时间: ${result.data.expire_time}秒`);
        } else {
            console.error('发送失败:', result.msg);
        }
        
        return result;
    } catch (error) {
        console.error('请求失败:', error);
        throw error;
    }
};

// 使用示例
sendSms('+1234567890', 'bind');
sendSms('+1234567890', 'login');
sendSms('+1234567890', 'forgot_password');
```
